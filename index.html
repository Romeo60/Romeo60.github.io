Portfolio

Here are some cool stuff I built for fun.

Bumphub.

My most loved project! Bumphub is a complete online store for upcoming artist to sell their music, be on charts, get likes for their songs while they earn real money royalties. It is a complete eCommerce site with an EFT payment gateway plugged in from payFast and my own API for voucher payments through kaCash.



kaCash.

KaCash is an online payment method that uses vouchers. A user simply buys a voucher from a dealer with the kacash app then proceed to enter the voucher at checkout from an online store. The app sends requests to a Rest API built in php on a remote server to generate and return a voucher code that will be used online.



Wika

My very own creation of a childhood sports game called Wika or natively "diWika", developed into a highscore based(infinity) web game. This HTML5 game is developed with javascript and the Phaser engine. Players tap the crates for the game charactor to kick the ball and score points by the ball succesfully hitting the tapped crate.



StudyBud.

This mobile app is designed for students. It is basically an educational tool that students(...and other organized people) can use to manage their daily lives. The app comes equiped with a notepad, voice note recorder, schedule planner and Motivation available everytime it's needed. Oh and study tips!
